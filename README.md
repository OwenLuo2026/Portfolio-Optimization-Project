# Portfolio Optimization Project

This project explores Modern Portfolio Theory and the relationship between risk and return.

## Key Topics

- Portfolio Variance
- Covariance Matrix
- Sharpe Ratio
- Efficient Frontier
- Capital Market Line
- Asset Allocation
- Investor Utility Functions

## Project Background

Completed as part of a Mathematics research project at University College London.

The project investigates how diversification and portfolio construction can improve risk-adjusted investment performance while accounting for different investor risk preferences.

## Industry Insights

The project was supplemented by discussions with professionals from Goldman Sachs Asset Management regarding portfolio construction and practical investment decision-making.
